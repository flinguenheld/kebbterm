### Installation

KebbTerm requires [Rust](https://www.rust-lang.org/).\
In your terminal, clone this repo:

```
git clone https://github.com/flinguenheld/kebbterm
```

Move into the folder:

```
cd kebbterm
```

Launch it and have fun:

```
cargo build --release && cargo run
```
